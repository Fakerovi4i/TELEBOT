## Структура проекта

### api_requests.py
- **Функция `get_move_by_name(move_name: str, page=1, limit=5)`**
  - ✅ УЖЕ возвращает НЕСКОЛЬКО фильмов (список), а не один
  - Возвращает `response.json().get("docs", [])` - это список всех найденных фильмов
  - Параметр `limit=5` означает максимум 5 результатов на страницу
  - Параметр `page` позволяет получать следующие страницы результатов
  - Функция работает корректно и возвращает множественные совпадения

### Как работает поиск фильмов
- API endpoint: `/movie/search`
- Параметры: `query` (название), `page` (номер страницы), `limit` (количество результатов)
- Ответ API содержит массив `docs` со всеми найденными фильмами
- Код уже настроен на возврат всех совпадений, а не одного фильма
