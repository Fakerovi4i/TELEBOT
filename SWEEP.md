## 📝 Заметки о коде

### Текущая структура бота (main.py)
- Использует aiogram 2.x (не 3.x!)
- База данных SQLite подключается при старте (`on_startup`)
- Таблица `users` для хранения данных пользователей
- Обработчик команды `/start` зарегистрирован

### Стиль кода
- Комментарии на русском языке
- Async/await для асинхронных функций
- Глобальные переменные для БД (учебный подход)
